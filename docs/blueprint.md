# **App Name**: RakshaConnect

## Core Features:

- Supplier Discovery: Allow buyers to discover verified suppliers through structured taxonomy and search.
- Supplier Profile Creation: Enable suppliers to create and manage their profiles, including capabilities and certifications.
- Taxonomy-Based Browsing: Implement a structured taxonomy to allow buyers to browse suppliers by major category, subcategory, and capabilities.
- Certification Filtering: Enable buyers to filter suppliers by certifications such as ISO, AS9100, and MIL.
- Supplier Saving: Allow buyers to save suppliers to their account for later viewing.
- Contact Supplier: Basic form enabling buyers to contact suppliers.
- AI Capability Suggester: Use AI to suggest relevant capabilities based on supplier's company information and selected taxonomy options; it serves as a tool to speed up and improve profile completion.

## Style Guidelines:

- Primary color: Dark blue (#243A73), reflecting a defense-grade, enterprise aesthetic.
- Background color: Very light gray (#F0F2F5), providing a neutral backdrop.
- Accent color: Mustard gold (#D4A743), used sparingly for key CTAs and highlights.
- Headline font: 'Space Grotesk', a sans-serif providing a computerized, techy feel. Body Font: 'Inter', for a modern, machined, objective, neutral look.
- Use clean, minimalist icons that are industry-relevant (e.g., gears, charts, certificates).
- Maintain a clean and structured layout with clear section hierarchy, following the Figma reference design.
- No flashy animations; use subtle transitions for a professional feel.
